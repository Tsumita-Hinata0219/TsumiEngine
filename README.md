# TsumiEngine
自作のDirectX12ベースゲームエンジンです

## 概要
TsumiEngine は C++ と DirectX12 を使った 3D/2D ゲームエンジンです
- 独自のリソース管理
- Lua スクリプト対応（sol2）
- レベルエディタ機能 などを含みます。

## 開発環境
- DirectX12
- Windows : 10.0.22621.0

## 外部ライブラリ
- assimp
- DirectXTex
- imgui
- nlohmann json

## 機能
- シーン管理
- 描画・テクスチャ管理
- Lua スクリプト拡張
